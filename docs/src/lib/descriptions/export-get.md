Exports a Markdown note as a self-contained, downloadable document. By default an HTML-comment metadata header (source path, timestamps, tags) is prepended to the note body; set the `Include-Metadata` header to `false` to return the body verbatim. Non-Markdown files are returned as a verbatim binary attachment. The response is served with a `Content-Disposition: attachment` header so browsers download it as a file.
